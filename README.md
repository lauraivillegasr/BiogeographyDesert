# Hierarchical Patterns of Soil Biodiversity in the Atacama Desert: Insights Across Biological Scales

Here you will find all the code implemented in the analysis of the manuscriot titled "Hierarchical Patterns of Soil Biodiversity in the Atacama Desert: Insights Across Biological Scales". The analysis is divided in two: Ecological estimates and Modeling. Within modeling you will find scripts for modeling using linear models and tandom forest.
